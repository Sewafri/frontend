## Design Context

**Register:** product (design serves the product — dashboards, tools, app UI)
**Brand Voice:** Clear, Capable, Trustworthy
**Design Direction:** Clean Slate

### Design Principles
1. **Clear authority** — The interface earns trust through precision: clean hierarchy, deliberate spacing, reliable interactions.
2. **African-first** — Designed for African learners — mobile-conscious, accessible, culturally relevant.
3. **Show, don't tell** — Visual feedback, progress indicators, data viz over walls of text.
4. **Practice what you preach** — The product teaches; the UI itself should be well-crafted and instructive.

### Key Visual Rules
- **Single accent** (Blue `#2563eb`) — CTAs, links, active states only — never decorative
- Flat surfaces with borders for separation, not shadows
- Gray-scale neutrals (cool), no warm tones, no orange
- One typeface (Geist) — hierarchy from size/weight/spacing
- No gradients, no glow effects, no decorative motion
